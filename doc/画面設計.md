# ジム利用記録システム 画面設計

利用者が操作するインターフェース（設置端末用）と、管理者が操作するWeb管理画面のレイアウトおよび遷移図です。

---

## 1. 画面遷移図

```mermaid
stateDiagram-v2
    [*] --> 待受画面 (U-01) : 起動

    %% 利用者フロー
    待受画面 (U-01) --> 学生証スキャン中 : 学生証をかざす (フェーズ2)
    学生証スキャン中 --> 確認画面 : スキャン成功 (キャッシュあり)
    学生証スキャン中 --> 手動入力画面 (U-04) : スキャン失敗 / キャッシュなし
    
    待受画面 (U-01) --> 手動入力画面 (U-04) : 画面をクリック (または学籍番号入力)
    手動入力画面 (U-04) --> 完了画面 (U-06) : 「チェックイン」ボタン押下
    確認画面 --> 完了画面 (U-06) : 「OK」ボタン押下
    
    手動入力画面 (U-04) --> 待受画面 (U-01) : キャンセル / 60秒無操作
    確認画面 --> 待受画面 (U-01) : キャンセル / 10秒無操作
    完了画面 (U-06) --> 待受画面 (U-01) : 3秒経過後に自動遷移

    %% 管理者フロー
    [*] --> Basic認証ダイアログ : /admin 以下にアクセス
    Basic認証ダイアログ --> 管理画面ダッシュボード (A-02) : ID・PWが一致
    Basic認証ダイアログ --> [*] : キャンセル / 認証失敗
```

* **Basic認証ダイアログ**: ブラウザ標準のユーザー名/パスワード入力画面が表示されるため、独自のログイン画面は実装しません。

---

## 2. 各画面のレイアウト設計

### 2.1 待受画面（U-01 / U-07）
利用者がジムに来た時に最初に目にする画面です。
オフライン時は、画面の右下に「オフライン動作中（未送信: X件）」とステータスが表示されます。

#### 【レイアウトイメージ（フェーズ1）】
```
+-----------------------------------------------------------------+
|                                                                 |
|                  WELCOME TO GYM RECORD SYSTEM                   |
|                                                                 |
|               [ 学籍番号を入力してチェックイン ]                |
|                                                                 |
|                     +---------------------+                     |
|                     |  学籍番号 (例: 12345) |                     |
|                     +---------------------+                     |
|                                                                 |
|                         [ 手動入力で進む ]                      |
|                                                                 |
|                                         [● ONLINE]              |
|                                         ※オフライン時は[● OFFLINE (未送信3件)]
+-----------------------------------------------------------------+
```

---

### 2.2 手動入力画面（U-04）
初回利用者や、画像スキャンがうまくいかなかった場合に入力する画面です。

```
+-----------------------------------------------------------------+

|                                                                 |
|                     ジム利用 登録フォーム                       |
|                                                                 |
|    学籍番号:   +------------------------------------------+     |
|                | 20261001                                 |     |
|                +------------------------------------------+     |
|                *入力すると氏名・クラスが自動で補完されます      |
|                                                                 |
|    氏名:       +------------------------------------------+     |
|                | 山田 太郎                                |     |
|                +------------------------------------------+     |
|                                                                 |
|    学科:       [ 建築学科           v ]  ※プルダウン           |

|                                                                 |

|                     |     チェックイン      |                   |
|                     +-----------------------+                   |
|                                                                 |
+-----------------------------------------------------------------+
```

---

### 2.3 管理画面ダッシュボード（A-02）
Basic認証成功後に遷移する管理者専用ポータルです。「利用ログ一覧」と「キャッシュ管理（学生マスタ情報）」をタブで切り替えられます。独自のログアウト処理は行わず、ブラウザセッションを終了することでログアウトと見なします。

```
+-----------------------------------------------------------------+
|  ジム利用記録 - 管理パネル                                      |
|  [利用ログ一覧]  [学生キャッシュ管理]                           |
```

本システムが提供する機能を分類して定義します。認証方式にBasic認証を採用したクラウド構成に適合させています。

---

## 1. 利用者（学生）向け機能

利用者がジム入室・退室時に操作、または視認する機能です。

| 機能ID | 機能名 | 機能概要 | 実装フェーズ |
| :--- | :--- | :--- | :--- |
| **U-01** | **初期（待受）画面表示** | 利用者を迎え入れる画面。画像認識（フェーズ2）または手動入力を促す。 | フェーズ1 |
| **U-02** | **学籍番号入力** | テンキーまたはキーボードから学籍番号を入力する。 | フェーズ1 |
| **U-03** | **キャッシュ自動補完** | 入力された学籍番号をもとにSupabase DBを検索し、一致するデータがあれば「氏名」「学科」「学年」「クラス」を自動入力する。教職員番号は `T` + 3桁で運用上新規発行され、先頭が `T` である場合は学科・学年入力を省略する。 | フェーズ1 |
| **U-04** | **手動入力（初回用）** | キャッシュがない場合、利用者が情報を画面上で選択・入力する。学科は`departments_master`からプルダウン選択。学年は選択した学科の修業年限（`years_count`）に応じて選択肢を自動絞り込み。クラスは`department_classes`からプルダウン選択。氏名はテキスト入力。教職員番号（`T` + 3桁）は先頭が `T` である場合に学科・学年・クラス入力を自動スキップし、初回登録フローにそのまま乗せる。手動入力画面で学籍番号が変更された場合は、変更後の番号を基準にキャッシュ検索・在室状態確認をやり直す。 | フェーズ1 |
| **U-05** | **チェックイン実行** | 入力データを確認し、チェックインボタンを押すことで利用ログを登録する。チェックイン要求時は、画面遷移時の判定結果に依存せず、送信された学籍番号でサーバー側が最新のキャッシュ検索・未退室チェックを再実行する。未退室のレコードが存在する場合は即時チェックアウトせず、チェックアウト確認画面へ遷移する。既存キャッシュが存在する学籍番号を初回登録として新規作成してはならない。 | フェーズ1 |
| **U-06** | **完了フィードバック表示** | 「チェックイン完了」または「チェックアウト完了」と氏名を画面に大きく表示し、3秒後に自動的に初期画面に戻る。 | フェーズ1 |
| **U-09** | **チェックアウト実行** | 学籍番号入力後、未退室のチェックインが検出された場合に表示される確認画面で、氏名・チェックイン時刻を確認してからチェックアウトを完了する。`usage_logs.checked_out_at` に退室時刻を記録する。 | フェーズ1 |
| **U-10** | **ランキング・統計表示** | 月間利用時間・連続利用日数ランキングを表示し、利用者・管理者が現状を把握できる。 | フェーズ1 |
| **U-11** | **自動補正通知** | 15時間超過で自動チェックアウトされた場合、次回アクセス時に「30分として記録した」旨の通知を表示し、未通知フラグを更新する。 | フェーズ1 |
| **U-07** | **学生証カメラスキャン** | PCのカメラに学生証をかざすことで、映像からOCRで学籍番号を自動読取する。 | **フェーズ2** |
| **U-08** | **手動切り替え（フォールバック）** | スキャンがうまくいかない場合、または手動入力を選びたい場合に、1タップで手動入力画面に遷移する。 | **フェーズ2** |

## 2. 管理者向け機能

ジムの運営管理者が、Webブラウザを介してデータの確認やシステムの運用保守を行うための機能です。
※管理者はVercelでホストされた管理者用URL（例：`/admin`）にアクセスして操作します。

| 機能ID | 機能名 | 機能概要 | 実装フェーズ |
| :--- | :--- | :--- | :--- |
| **A-01** | **管理者ログイン認証（Basic）** | `/admin` アクセス時にNext.js Middlewareがリクエストを検証し、ブラウザ標準のBasic認証ダイアログを表示。ユーザー名/パスワードはVercelの環境変数（`ADMIN_USER` / `ADMIN_PASSWORD`）で管理する。独自のログイン画面やセッション管理の実装は省略する。 | フェーズ1 |
| **A-02** | **利用ログ一覧表示** | これまでの利用ログを一覧で確認できる。日付範囲・学籍番号・氏名・学科・学年・クラスで絞り込み・検索が可能。論理削除されていないレコード（`deleted_at IS NULL`）のみ表示する。 | フェーズ1 |
| **A-03** | **利用ログCSVエクスポート** | 表示されている利用ログ、または指定期間の全利用ログをCSV形式でダウンロードする。チェックイン時刻・チェックアウト時刻・滞在時間を含む。 | フェーズ1 |
| **A-04** | **学生キャッシュ管理** | 登録されている学生キャッシュ（補完データ）の一覧を表示し、誤った情報の修正・論理削除を行う。削除したレコードはゴミ箱に30日間保持され復元可能。毎年4月に管理画面側のサーバー処理で有効な学生キャッシュを一度だけ自動進級し、所属学科の修業年限に達している最終学年の学生キャッシュは論理削除する。教職員および論理削除済みデータは対象外とする。 | フェーズ1 |
| **A-05** | **リアルタイム利用状況確認** | 本日の利用人数や、現在ジムにいると想定される在室人数（`checked_out_at IS NULL` のレコード数）をダッシュボード形式で確認できる。 | フェーズ1 |
| **A-06** | **利用ログ論理削除・ゴミ箱・復元** | 個別または一括で利用ログを論理削除する（確認ダイアログ付き）。削除されたレコードはゴミ箱タブで確認・復元できる。30日経過後に物理削除される。 | フェーズ1 |
| **A-07** | **マスタ管理（学科・クラス）** | `departments_master`（学科名・修業年限・表示順）および `department_classes`（クラス名・対象学年・表示順）のCRUD操作を行う管理画面。学科削除は即時の物理削除ではなく論理削除とし、紐づくクラスも同時にゴミ箱へ移動する。削除後30日以内はゴミ箱から復元でき、30日経過またはゴミ箱で完全削除した場合に物理削除する。入力フォームの選択肢には論理削除されていない学科・クラスのみ即時反映される。 | フェーズ1 |
| **A-08** | **多言語切替・統計タブ** | 管理画面で利用統計・ランキングを確認できるほか、画面メッセージの日本語/英語切替に対応する。 | フェーズ1 |

## 3. システム共通・バックエンド機能

画面の裏側で動作する、データの永続化やネットワーク連携を行う機能です。

| 機能ID | 機能名 | 機能概要 | 実装フェーズ |
| :--- | :--- | :--- | :--- |
| **S-01** | **オンライン時直接登録** | ネットワーク接続がある場合、入力された利用ログを即時Supabase PostgreSQLに登録し、学生キャッシュを必要に応じて作成する。登録前には、送信された学籍番号を正規化したうえで `users_cache` と未退室 `usage_logs` を再検索し、既存学籍番号の二重登録を防止する。`users_cache.student_id` は一意であり、同一学籍番号の学生キャッシュを複数作成してはならない。 | フェーズ1 |
| **S-02** | **オフライン時一時保存** | ネットワーク切断時、フロントエンドで検知し、ログデータをブラウザの IndexedDB に一時保存する。チェックイン・チェックアウトどちらも保存可能。 | フェーズ1 |
| **S-03** | **オンライン復帰検知と自動同期** | ブラウザがネットワーク復旧（`online` イベント）を検知すると、IndexedDB にある未同期ログをSupabaseに自動で順次同期する。 | フェーズ1 |
| **S-04** | **未同期ログの件数通知** | オフライン時に一時保存された未同期ログがブラウザ内に何件あるかを、チェックイン画面の片隅に小さく表示し、管理者や設置者が状況を把握できるようにする。 | フェーズ1 |
| **S-05** | **自動画面初期化（タイムアウト）** | 手動入力画面で60秒間無操作だった場合、画面占有を防ぐために自動的に初期（待受）画面へ戻る。個人情報の画面残留防止も兼ねる。 | フェーズ1 |
| **S-06** | **自動チェックアウト（15時間）** | チェックイン後15時間が経過しても `checked_out_at` が記録されていない場合、APIまたはバッチ処理が `checked_out_at = checked_in_at + 15h` を自動的にセットする。利用時間は実際の経過ではなく一律30分として保存し、`is_adjusted = true` / `is_notified = false` を付与する。在室人数の正確な把握のために機能する。 | フェーズ1 |
